## はじめに
これは「オセロ」ゲームのリポジトリです。

## コンセプト
「立体４目並べ」では、盤面を配列で実装したため処理の負荷が大きくなっていたため、  
「オセロ」では盤面をビットボードで実装することで処理の負荷を小さくして最大探索局面数の向上を目指す。

## 仕様
- 思考ルーチン（COM）を実装
- Mini-Max法、αβ法を用いて思考ルーチンを設計
- 最大探索局面数：１５００万局面
- 定石を実装
- ルール：通常のオセロと同じ
